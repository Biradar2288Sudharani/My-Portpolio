Latest project

Face Recognition Attendance System

This project is a desktop-based Python application developed using OpenCV and Tkinter. It automates the process of taking student attendance using real-time face recognition.
The system detects and recognizes registered student faces through webcam input. Attendance is marked automatically and saved in a CSV file and MySQL database.
Admin can view, search, and update attendance records via a GUI panel. The system includes Twilio API integration to send WhatsApp alerts to parents.

Pandharpur_Guide

“Pandharpur_Guide” is a live web application project designed and developed with the aim of helping pilgrims and tourists visiting the holy town of Pandharpur, Maharashtra. The platform offers comprehensive and user-friendly guidance to travelers by showcasing essential facilities such as accommodation (hotels, lodges), restaurants, temple routes, local transport, and reviews. This project integrates real-time data presentation using Django (Python Web Framework), SQLite3 database, HTML, CSS, JavaScript, and responsive design techniques. It was developed to serve both local and remote users, enabling them to pre-plan their journey or navigate efficiently while visiting. The goal was to combine technology and spiritual tourism, easing the experience of devotees and travelers by providing authentic and structured information through a centralized platform.

Vehicle Detection

This project is a Python-based application that detects moving vehicles using OpenCV. It uses video or webcam input to identify and track objects in real time.
Background subtraction and contour detection techniques are applied for motion analysis. Bounding boxes are drawn around detected vehicles as they move across the screen.
The system can be extended for traffic surveillance, vehicle counting, or smart signals.
